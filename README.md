# 🏝️ Island

> **"Your island. Your rules. Your freedom."**

Island is a free, open-source, privacy-first desktop messenger and file vault.  
No servers. No accounts. No phone number. No surveillance.  
**Like Bitcoin — no one owns Island. No one controls it.**

---

## 📥 Download & Install

Go to **[Releases](https://github.com/omarbouzganda/island/releases)** → download for your OS → double click.

| OS | File | Steps |
|----|------|-------|
| 🪟 Windows | `Island-Setup.exe` | Double click → Next → Done |
| 🐉 Linux / Kali | `Island.AppImage` | Right click → Allow executing → Double click |
| 🍎 macOS | `Island.dmg` | Open → Drag to Applications |

**That's it. No terminal. No Node.js. No setup.**

## 🔒 Privacy

- Your ID is generated **locally** — no account needed
- Messages are **end-to-end encrypted**
- Everything stored in an **encrypted local vault**
- A free relay helps users connect — it **never sees messages**
- **Local network mode** — talk with zero internet on same WiFi

---

## 🌉 Dual Boot Bridge

Running Kali + Windows on the same PC?

1. Open **Bridge** page in Island
2. Click **Scan partitions**
3. Island finds your shared partitions automatically
4. Mount → drop files → boot other OS → files are there

No USB. No internet. No setup.

## 🛠️ For developers — build from source

```bash
git clone https://github.com/omarbouzganda/island
cd island
npm install
npm start
```

Build installers:
```bash
npm run build:win      # → Island-Setup.exe
npm run build:linux    # → Island.AppImage
npm run build:mac      # → Island.dmg
```
